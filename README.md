# calculator
A simple calculator parser for PLUS and Minus

Open the **CalculatorParser.sln** file with Visual Studio.
Build the solution. 

Open a command window.
Go to the CalculatorParser\bin\Release folder.
Run the application : 
```
CalculatorParser.exe [input]
```
where [input] specifies the string to parse.

Ex: CalculatorParser.exe "-1+4 - 6 + 7"
